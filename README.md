🌃 City Tours

is a full-stack Java-based web application that allows travelers to browse, book, and manage city tours.



 📌 Features

- Browse available city tours
- Book tours 
- Admins can manage bookings (CRUD), each limited to their own city


## 🛠️ Technologies Used

- Java
- Spring Framework
- Spring Security
- JPA (Hibernate)
- Bootstrap
- JavaScript
